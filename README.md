# C++ Password Manager

A command-line password manager written in C++.  
This project was built to practice secure data handling, modular C++ architecture, and CLI-based application design.

## Features

- Secure password storage
- Vault system for managing credentials
- Password generator
- Command-line interface
- Modular project structure
- JSON-based data storage

## Technologies

- C++
- STL
- CMake
- JSON library (nlohmann/json)
- sodium (cryptography library)
- Argon2 (password hashing)

## What I Learned

This project helped me practice:

- structuring larger C++ projects
- separating logic into modules
- file handling and data serialization
- building command-line tools
